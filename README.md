# ovgenpy
Python program to render wave files into oscilloscope views, featuring improved correlation-based triggering algorithm

## Dependencies

- Python 3.6 or higher
- FFmpeg

## Installation

Virtualenv:

```shell
# create virtualenv
pip install -e .
```
